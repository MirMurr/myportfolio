Payapi multipage website

This project was designed by frontendmentor. The website contains a contact form for help requests and all pages contain a short contact form where the user can input their email address. The 2 types of forms will be submitted to separate google sheets. I have written a regex in JS to validate the email inputs. I used HTML, CSS, vanilla JS, API script (created by Jamie Wilson) to build this project. The layout is fully responsive. For more info please read the README file in the github repo.
